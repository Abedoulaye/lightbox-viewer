# [Lightbox Image Gallery]

A responsive image gallery with a JavaScript lightbox overlay.

## Demo

![https://abedoulaye.github.io/lightbox-viewer/]

## Features

- Click any gallery thumbnail to open a full-size lightbox view
- Close the lightbox via the X button or by clicking the dark background
- Fully responsive layout
- Clean, minimal design

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
